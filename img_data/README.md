The raw image data is the dataset `01_LE` from the publicly available data repository of Badian et al. [1]. The result files (mosaic images and SQLite database file) were created with _HIT_ using the default parameter values (which can be reset by clicking the button "Defaults for 2D datasets" in the parameter dialog).
